# GenAI-Powered Retail Sales Analytics Chatbot

This project implements a **Generative AI powered conversational analytics system** for retail sales data.

The system allows users to interact with retail data using natural language queries and obtain business insights in real time.

## Features

- Conversational AI chatbot for retail analytics
- Sales trend analysis
- Product performance insights
- AI-generated business explanations
- Interactive dashboard
- Suggested query panel
- CSV and PDF report export
- Metadata transparency (timestamps and response time)

## Technologies Used

- Python
- Streamlit
- Pandas
- SQLite
- Groq LLM (LLaMA 3)
- Matplotlib
- Seaborn

## System Architecture

User Query → Chatbot Interface → Analytics Engine → AI Model → Insights & Visualization
